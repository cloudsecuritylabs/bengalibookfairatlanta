## How to Add Portrait Photos
Replace any portrait placeholder by placing the correctly named .jpg in the right folder.
The website will automatically display it.

## Publishing to GitHub Pages
1. Push this entire folder to your GitHub repository
2. Go to Settings → Pages → select branch → Save
3. Your site will be live at: https://YOUR-USERNAME.github.io/REPO-NAME/
